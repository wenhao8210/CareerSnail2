# Changelog

## 2026-04-20

- 简历能力画像分析固定使用 OpenAI GPT-5，并移除了前端模型选择入口。
- 能力画像问答精简为 3 题，并按不同岗位动态调整第三题内容。
- 能力画像结果页新增 5 秒后弹出的模拟面试引导，可一键跳转并自动生成题库。
- 模拟面试支持接收能力画像页传入的简历内容与岗位方向，自动创建题库。
- 小蜗日程默认切换为 `Overview` 首屏展示。
- 面试相关统计从小蜗日程迁移到面试复盘页，新增 `Analysis` 视图、待面试/已通过双 bucket、拖拽归类和转化率展示。
- 面试复盘页新增 `待记录` 列表，可基于 `#面试` 标签任务直接创建复盘。
- 面试复盘新建记录支持上传 `.txt`，并可一键整理为逐条 Q&A。
- 上传的作品集文件现在会同步上传到 Supabase Storage，并返回公开链接。
- 暂时下线蜗牛岛功能，移除了页面、同步接口和所有导航入口。
